## Packages
react-qr-code | Display WhatsApp connection QR code
react-markdown | Render analysis output from LLM
date-fns | Formatting log and import timestamps
lucide-react | Icons (already installed but crucial)

## Notes
- Backend is assumed to run on the same origin (no absolute URLs needed for API calls).
- The file upload endpoint expects standard `multipart/form-data`.
- Strictly dark theme applied via standard CSS variables in index.css.
- Using Google Fonts: 'Outfit' for headings and 'Plus Jakarta Sans' for body text to give a modern, premium feel.
